Seismic Noise Removal Tutorial

Learn how to effectively use our seismic data filtering tool

1

Access the Filter Tool

Navigate to the "Filter" page using the navigation menu. This will load our Streamlit application in an embedded frame.

// Click on the "Filter" link in the navigation bar
2

Input Seismic Data Parameters

Enter the details of the seismic data you want to process:

  • Network Code - The seismic network identifier (e.g., XB)
  • Station Code - The specific station code (e.g., ELYSE)
  • Location Code - Location identifier (e.g., 02)
  • Channel - Component channels (use ? as wildcard, e.g., BH?)
  • Start Time - UTC start time of the data window
  • End Time - UTC end time of the data window
!

Important: Three-Component Data Requirement

For the polarization filter to work correctly, you must select data that includes all three components (typically Z, N, E or 1, 2, 3).

The application expects exactly 3 traces. If you don't select data with 3 components, the filtering process will fail.

Make sure your channel selection includes all three components. Using a wildcard (like "BH?") often helps retrieve all available components.

3

Fetch and Process Data

Click the "Fetch & Process Data" button to:

  1. Retrieve data from the IRIS database
  2. Preprocess the waveforms (remove offsets, drifts, and taper edges)
  3. Apply band-pass filters to keep useful frequencies
  4. Use polarization to suppress random noise
  5. Visualize the difference between raw and cleaned signals
4

Interpret Results

After processing, you'll see:

  • A comparison plot showing raw data (top) and cleaned data (bottom)
  • The ability to download the cleaned data as a CSV file

Examine the cleaned signal for seismic events that were previously obscured by noise.

5

Troubleshooting

If you encounter errors:

  • Verify your network, station, and channel codes
  • Ensure you've selected a time range with available data
  • Confirm that your selection includes exactly three components
  • Check your internet connection (data is fetched from IRIS)

Ready to try it out?

Now that you understand how to use the seismic noise removal tool, navigate to the Filter page to begin processing your data.
